# AlviElysium

Platform digital yang dirancang untuk menciptakan pengalaman membaca yang mendalam, menginspirasi, dan memikat.

## Client (Nuxt.js v2)
### Install
```shell
$ yarn install
```
### Build
```shell
$ yarn run build
```
### Start
```shell
$ yarn run start
```
### Dev
```shell
$ yarn run dev
```
### Generate
```shell
$ yarn run generate
```
